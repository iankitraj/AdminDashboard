# Admin Dashboard

A modern and scalable Admin Dashboard built with Next.js, TypeScript, and Tailwind CSS following industry-level frontend architecture practices.

---

## 🚀 Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* ESLint

---

## 📁 Folder Structure

```bash
src/
├── app/
├── components/
├── constants/
├── features/
├── hooks/
├── lib/
├── providers/
├── services/
├── store/
├── styles/
├── types/
└── utils/
```
